**GET /api/v1/profit/summary** implementation:
1. Query trade_records with aggregate: group by ea_id, compute:
   - `total_spent`: `func.sum(case((TradeRecord.outcome == "bought", TradeRecord.price), else_=0))`
   - `total_earned_gross`: `func.sum(case((TradeRecord.outcome == "sold", TradeRecord.price), else_=0))`
   - `trade_count`: `func.count(TradeRecord.id)`
2. For each ea_id row, compute in Python:
   - `total_earned = int(total_earned_gross * (1 - EA_TAX_RATE))`  (apply 5% EA tax)
   - `net = total_earned - total_spent`
3. Look up player names: join with PlayerRecord on ea_id, or do a separate query. Use LEFT JOIN so records without a PlayerRecord still appear (use ea_id as fallback name).
4. Build per_player list: `[{"ea_id": int, "name": str, "total_spent": int, "total_earned": int, "net_profit": int, "trade_count": int}]`
5. Build totals by summing per_player values
6. Return:
```json
{
  "totals": {
    "total_spent": int,
    "total_earned": int,
    "net_profit": int,
    "trade_count": int
  },
  "per_player": [...]
}
```

Add endpoint:
```python
@router.delete("/portfolio/{ea_id}")
async def delete_portfolio_player(
    request: Request,
    ea_id: int = Path(..., description="EA ID of the player to remove"),
    budget: int = Query(..., gt=0, description="Total portfolio budget in coins"),
):
```

Implementation:
1. Look up PortfolioSlot by ea_id. If not found, return HTTPException(404, "Player not in portfolio").
2. Get the freed budget: `freed_budget = slot.buy_price`
3. Cancel pending/in-progress actions for this ea_id (per D-08):
   ```python
   await session.execute(
       update(TradeAction)
       .where(TradeAction.ea_id == ea_id, TradeAction.status.in_(["PENDING", "IN_PROGRESS"]))
       .values(status="CANCELLED")
   )
   ```
4. Delete the PortfolioSlot:
   ```python
   await session.execute(
       delete(PortfolioSlot).where(PortfolioSlot.ea_id == ea_id)
   )
   ```
5. Get remaining portfolio slot ea_ids:
   ```python
   remaining_result = await session.execute(select(PortfolioSlot.ea_id))
   remaining_ea_ids = {row[0] for row in remaining_result.all()}
   ```
6. Run optimizer for replacements (per D-07):
   - Query latest viable PlayerScore + PlayerRecord (reuse the same subquery pattern from `get_portfolio()`)
   - Build scored entries via `_build_scored_entry()` for candidates NOT in remaining_ea_ids
   - Call `optimize_portfolio(scored_candidates, freed_budget)`
   - The returned list = replacement players
7. Commit.
8. Return:
   ```json
   {
     "removed_ea_id": ea_id,
     "freed_budget": freed_budget,
     "replacements": [
       {"ea_id": int, "name": str, "buy_price": int, "sell_price": int, "margin_pct": int, "expected_profit_per_hour": float}
     ]
   }
   ```

Note: `optimize_portfolio()` mutates dicts. Build fresh dicts via `_build_scored_entry()` per the existing pattern (portfolio.py line 101 comment).
