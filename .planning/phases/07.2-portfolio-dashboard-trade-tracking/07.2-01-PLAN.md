    **Query logic (3 queries inside one session):**

    Query 1 — All portfolio slots:
    ```python
    slots_stmt = select(PortfolioSlot.ea_id, PortfolioSlot.buy_price, PortfolioSlot.sell_price)
    ```

    Query 2 — Trade record aggregation per ea_id (same pattern as profit.py):
    ```python
    agg_stmt = select(
        TradeRecord.ea_id,
        func.sum(case((TradeRecord.outcome == "sold", TradeRecord.price), else_=0)).label("total_sold_gross"),
        func.sum(case((TradeRecord.outcome == "bought", TradeRecord.price), else_=0)).label("total_bought"),
        func.count(case((TradeRecord.outcome == "sold", 1))).label("times_sold"),
        func.count(case((TradeRecord.outcome == "bought", 1))).label("bought_count"),
        func.count(case((TradeRecord.outcome == "expired", 1))).label("expired_count"),
    ).group_by(TradeRecord.ea_id)
    ```

    Query 3 — Most recent outcome per ea_id (use MAX(id), NOT MAX(recorded_at) — Pitfall 3):
    ```python
    latest_record_subq = (
        select(TradeRecord.ea_id, func.max(TradeRecord.id).label("max_id"))
        .group_by(TradeRecord.ea_id)
        .subquery()
    )
    status_stmt = (
        select(TradeRecord.ea_id, TradeRecord.outcome)
        .join(latest_record_subq,
              (TradeRecord.ea_id == latest_record_subq.c.ea_id)
              & (TradeRecord.id == latest_record_subq.c.max_id))
    )
    ```

    Query 4 — Latest MarketSnapshot per ea_id for current_lowest_bin:
    ```python
    latest_snapshot_subq = (
        select(MarketSnapshot.ea_id, func.max(MarketSnapshot.id).label("max_id"))
        .group_by(MarketSnapshot.ea_id)
        .subquery()
    )
    bin_stmt = (
        select(MarketSnapshot.ea_id, MarketSnapshot.current_lowest_bin)
        .join(latest_snapshot_subq,
              (MarketSnapshot.ea_id == latest_snapshot_subq.c.ea_id)
              & (MarketSnapshot.id == latest_snapshot_subq.c.max_id))
    )
    ```

    Query 5 — Player names (same pattern as profit.py):
    ```python
    name_stmt = select(PlayerRecord.ea_id, PlayerRecord.name).where(
        PlayerRecord.ea_id.in_(ea_ids)
    )
    ```

    **Assembly logic:**
    - For each PortfolioSlot ea_id, build a player dict
    - Status: look up in latest_outcome_map, apply OUTCOME_TO_STATUS, default to "PENDING"
    - Realized profit per player: `int(total_sold_gross * (1 - EA_TAX_RATE)) - total_bought` (EA tax in Python, same as profit.py)
    - Unrealized P&L: only for BOUGHT or LISTED status: `current_lowest_bin - buy_price`. Set to None for PENDING, SOLD, EXPIRED.
    - current_bin: from MarketSnapshot lookup, None if no snapshot exists
    - Summary: sum realized_profit across all players, sum unrealized_pnl for held players only, trade_counts from aggregation
