**Helper function for readability:**
```python
async def seed_slot(client, ea_id, buy_price, sell_price, name="Test Player"):
    resp = await client.post("/api/v1/portfolio/slots", json={"slots": [
        {"ea_id": ea_id, "buy_price": buy_price, "sell_price": sell_price, "player_name": name}
    ]})
    assert resp.status_code == 201
    return resp

async def get_pending(client):
    resp = await client.get("/api/v1/actions/pending")
    assert resp.status_code == 200
    return resp.json()["action"]

async def complete_action(client, action_id, price, outcome):
    resp = await client.post(f"/api/v1/actions/{action_id}/complete",
                             json={"price": price, "outcome": outcome})
    assert resp.status_code == 200
    return resp.json()
```

**CRITICAL for lifecycle tests:** Always extract `action["id"]` from GET /pending response and use it in complete_action call. Never hardcode action IDs.
