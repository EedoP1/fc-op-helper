    **async def run_portfolio(url: str, budget: int) -> None:**
    - Make GET request to `{url}/api/v1/portfolio?budget={budget}` using `httpx.AsyncClient(timeout=30.0)`
    - On `httpx.ConnectError`: print `f"Error: Cannot reach server at {url}. Start the backend with: uvicorn src.server.main:app"` and `sys.exit(1)` (per D-07)
    - On non-2xx response: print `f"Error: Server returned {resp.status_code}"` with response text snippet, exit 1
    - Parse JSON response, map "data" list to display format
    - Field mapping from API response to display dict:
      ```python
      mapped = {
          "player_name": item["name"],
          "rating": item["rating"],
          "position": item["position"],
          "buy_price": item["price"],       # API key is "price"
          "margin_pct": item["margin_pct"],
          "op_ratio": item["op_ratio"],
          "expected_profit": item["expected_profit"],
          "efficiency": item["efficiency"],
      }
      ```
    - Call `display_results(mapped_list, budget, budget_used)` where budget_used comes from response["budget_used"]
    - Call `export_csv(mapped_list)` and print path (per D-06)
