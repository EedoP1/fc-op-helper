## Decisions Made

- `conftest.py` passes `DATABASE_URL` via subprocess env to the uvicorn process. The harness does not need to set it — `src.config` reads `DATABASE_URL` at import time from `os.environ`.
- `real_ea_id` fixture queries asyncpg directly. Returns `None` on a fresh empty test container — tests that need real players use `POST /portfolio/generate` instead.
- `app.state.read_session_factory = session_factory` preserves the attribute name used by all API endpoints without modifying them. Postgres MVCC makes separate read/write engines unnecessary.
- Migration script uses `docker run dimitri/pgloader` (not native pgloader) because pgloader has no Windows binary.
