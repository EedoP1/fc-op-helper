<objective>
Retain raw market data per player for 1 month instead of discarding it after scoring.

Purpose: Currently `scan_player()` fetches `PlayerMarketData` (sales, price history, live auctions), scores it, then discards the raw data. To enable future scoring improvements (backtesting, trend analysis, better OP detection), we need to persist the raw market snapshot from each scan and retain it for 30 days.

Output: New DB tables for market snapshots with sales and price history, scanner writes raw data alongside scores, daily cleanup purges data older than 30 days.
</objective>

   # Persist raw market snapshot
   if market_data is not None:
       snapshot = MarketSnapshot(
           ea_id=ea_id,
           captured_at=now,
           current_lowest_bin=market_data.current_lowest_bin,
           listing_count=market_data.listing_count,

               sold_price=sale.sold_price,
           ))

       for point in market_data.price_history:
           session.add(SnapshotPricePoint(
               snapshot_id=snapshot.id,
               recorded_at=point.recorded_at,
               lowest_bin=point.lowest_bin,
           ))

   Note: The existing `await session.flush()` on line 338 and `await session.commit()` in `_classify_and_schedule` will handle persistence. Add the `await session.flush()` after `session.add(snapshot)` to obtain `snapshot.id` before creating child rows.

3. Add a new method `run_cleanup()` to `ScannerService`:

   ```python
   async def run_cleanup(self) -> None:
       """Delete market snapshots older than MARKET_DATA_RETENTION_DAYS.

       FK cascade on SnapshotSale and SnapshotPricePoint ensures child
       rows are deleted automatically. Also prunes old PlayerScore rows
       beyond retention to keep the DB lean.
       """
       cutoff = datetime.utcnow() - timedelta(days=MARKET_DATA_RETENTION_DAYS)
       async with self._session_factory() as session:
           # Delete old snapshots (cascades to sales + price points)
           from sqlalchemy import delete
           result = await session.execute(
               delete(MarketSnapshot).where(MarketSnapshot.captured_at < cutoff)
           )
           snapshot_count = result.rowcount

           # Also prune old PlayerScore rows
           result = await session.execute(
               delete(PlayerScore).where(PlayerScore.scored_at < cutoff)
           )
           score_count = result.rowcount

           await session.commit()
       logger.info(
           f"Cleanup: deleted {snapshot_count} snapshots and {score_count} scores older than {MARKET_DATA_RETENTION_DAYS} days"
       )
   ```
